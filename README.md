<h2>Personal Portfolio Site</h2>

<p>This is a professional personal portfolio website built using HTML and CSS. It features a clean landing page with a hero section introducing the creator, a Projects section showcasing key projects like Art Mart, Rock-Paper-Scissors game, and a Calculator, and a Contact section with email and LinkedIn information. The site is fully responsive using CSS media queries, ensuring it looks good on both desktop and mobile devices. The design focuses on simplicity, readability, and highlighting important projects for potential employers</p>
